# Java-FSHN

## Purpose

This repository contains Java programming exercises and examples created as part of coursework at **FSHN** (Faculty of Natural Sciences). It covers introductory Java topics including static methods, object-oriented programming, and graphical user interfaces (GUI) using AWT and Swing.

## Programming Language

- **Java**

## Project Structure

| Folder | Description |
|--------|-------------|
| `Laborator1/` | Laboratory 1 — exercises on static methods and OOP class design |
| `Laborator2/` | Laboratory 2 — Java Swing GUI exercises (JFrame, dialogs, login) |
| `LeksionSeminar1/` | Lecture/Seminar 1 — static methods: temperature, parity, factorial |
| `LeksionSeminar2/` | Lecture/Seminar 2 — Java GUI programming with AWT and Swing |

## How to Compile and Run

Each Java source file can be compiled and run from the command line:

```bash
# Compile
javac FileName.java

# Run
java ClassName
```

> **Note:** Files using Swing or AWT require a graphical display environment to run.
