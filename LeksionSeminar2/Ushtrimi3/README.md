# Ushtrimi 3 — Counter Application

## Overview

This folder contains a simple **counter application** built with Java Swing. The user clicks a button to increment a counter, and the current value is displayed in a text field.

## Files

| File | Description |
|------|-------------|
| `Ushtrimi3.java` | Swing-based counter with a button and a read-only text field |

## What the Code Demonstrates

- Using a `JButton` and `JTextField` together
- Making a text field read-only with `setEditable(false)`
- Maintaining state (the `counter` variable) across button clicks
- Using an anonymous `ActionListener` to handle events
- Updating a UI component (`JTextField`) when the button is clicked

## How to Run

```bash
javac Ushtrimi3.java
java Ushtrimi3
```
