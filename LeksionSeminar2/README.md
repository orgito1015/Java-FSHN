# LeksionSeminar 2 — Lecture/Seminar 2: Java GUI Programming

## Overview

This folder contains **Seminar 2** exercises and examples for the Java course at FSHN. The topic is **Java Graphical User Interface (GUI) programming** using the **AWT** (Abstract Window Toolkit) and **Swing** libraries.

The full exercise sheet is available in `Seminar 2.pdf`.

## Folder Structure

| Folder | Description |
|--------|-------------|
| `Ushtrimi1/` | Basic GUI examples — creating a simple window with a label |
| `Ushtrimi2/` | Form input exercise — text field, label, and button |
| `Ushtrimi3/` | Counter application using Swing |
| `Ushtrimi4/` | Running sum calculator using Swing |
| `UshtrimeSeminar2/` | Additional seminar exercise slots (numbered subfolders) |

## Concepts Demonstrated

- Creating windows with `Frame` (AWT) and `JFrame` (Swing)
- Adding labels, text fields, and buttons to a window
- Handling button click events with `ActionListener`
- Updating UI components in response to user input
- Difference between AWT and Swing approaches
