# 📚 Laboratori 2 - Java (Swing Exercises)

## 📌 Përshkrimi
Ky repository përmban një seri ushtrimesh praktike në Java Swing, të fokusuara në krijimin e ndërfaqeve grafike (GUI) dhe përdorimin e komponentëve bazë si:

- JFrame
- JLabel
- JButton
- JTextField
- JOptionPane

Qëllimi është të kuptohen konceptet bazë të zhvillimit të aplikacioneve desktop në Java.

---

## 🛠️ Teknologjitë e përdorura

- Java (JDK 8+)
- Java Swing (GUI framework)

---

## 📂 Struktura e Projektit

Çdo ushtrim është i implementuar si klasë e veçantë:

