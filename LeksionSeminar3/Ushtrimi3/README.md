# Ushtrimi 3 — Numërim i Karakterit në Tekst

## Detyrë

Shkruani një aplikacion në JAVA që merr **një rresht teksti** dhe **një karakter** nëpërmjet dy fushave tekst, dhe në klikim të një butoni gjen **sa herë ndodhet ky karakter** në tekst.

## Skedarët

| Skedar | Përshkrim |
|--------|-----------|
| `Ushtrimi3.java` | Aplikacion Swing me dy JTextField dhe buton numërimi |

## Çfarë demonstron kodi

- Dy `JTextField` për input të ndryshëm (tekst + karakter)
- Validim i gjatësisë me `String.length()`
- Lexim karakteri me `String.charAt(0)`
- Numërim me cikel `for` dhe krahasim karakteresh
- Shfaqje rezultati dinamik në `JLabel`

## Si ekzekutohet

```bash
javac Ushtrimi3.java
java Ushtrimi3
```
