# Ushtrimi 5 — Makinë Llogaritëse

## Detyrë

Ndërtoni një aplikacion në JAVA që krijon një **makinë llogaritëse** me katër veprime bazë.

## Skedarët

| Skedar | Përshkrim |
|--------|-----------|
| `Ushtrimi5.java` | Makinë llogaritëse Swing me ekran dhe tastierë numerike |

## Çfarë demonstron kodi

- Organizim me `GridLayout` për tastierën numerike
- Ekran (JTextField jo i editushëm) me font të madh
- Ruajtje gjendjes me variabla statike (`numri1`, `operacioni`, `inputiRi`)
- Kontroll i pikës dhjetore (nuk lejohen dy pika)
- Trajtim i ndarjes me zero me mesazh gabimi
- Butoni "C" (clear) për risetim të gjendjes
- Formatim inteligjent i rezultatit (numër i plotë kur është e mundur)

## Si ekzekutohet

```bash
javac Ushtrimi5.java
java Ushtrimi5
```
